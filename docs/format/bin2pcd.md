# bin2pcd

## 功能

- 将bin格式点云转换为pcd格式点云
- 支持文件夹批量转换

## 提供一种使用模式

- 输入参数

### 输入参数的使用例子

```bash
apk format bin2pcd --input ./bins --input_dims xyz --output ./pcds

apk format bin2pcd --input ./bins --input_dims xyzi --output ./pcds

apk format bin2pcd --input ./bins --input_dims xyzir --output ./pcds
```
